* LCD SPI

** protocol
write data:
- 0x0
- data

read data:
- 0x40
- 0x0 <= read

write command:
- 0x80
- command

read status:
- 0xC0
- 0x0 <= read

write reg:
- write command reg
- write data val

read reg:
- write command reg
- return read data 

write reg:
- 0x80
- reg
- 0x0
- val
